Help & Instructions

Learn how to use the Test Portal effectively

Getting Started

  1. Browse available tests from the "Available Tests" section
  2. Click on a test to start taking it
  3. Answer all multiple-choice questions
  4. Submit your test to see your results
  5. Review your performance in the "Results" section

Creating Custom Tests

You can create your own tests:

  1. Click "Create New Test" in the Available Tests section
  2. Enter test details (title, description, time limit)
  3. Add questions with 4 options each
  4. Mark the correct answer for each question
  5. Save your test to make it available

Customization Options

Personalize your portal:

  • Change themes (Light, Dark, Blue, Green)
  • Toggle auto-next question feature
  • Show/hide timer during tests
  • Enable/disable instant feedback
  • Change your display name and avatar color
